# MGraph.js

## Description

A graph lib implemented in JavaScript.
 
## Implemented

- addVertex
- removeVertex
- getVertex
- existsVertex
- setVertexProperties
- findVertices
- addEdge
- removeEdge
- getEdge
- existsEdge
- setEdgeProperties
- getAdjacents
- getAdjacencyList.
